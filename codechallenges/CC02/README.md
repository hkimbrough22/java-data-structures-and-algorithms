#Array Insert-Shift

Takes in an array and a value to be added. Without utilizing any of the built-in methods available to your language, return an array with the new value added at the middle index.
## Example
###Input
```javascript
[2,4,6,-8], 5


[42,8,15,23,42], 16	
```
### Output
```Javascript
[2,4,5,6,-8]


[42,8,15,16,23,42]
```
